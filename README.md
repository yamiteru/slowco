# Slowco

- Komunikace 1:1
- Nové zprávy dostanu 1x denně ráno (7:00)
- Nové zprávý můžu psát kdykoli přes den
- Povolit 1 zprávu na příjemce denně (7:00)
- Služba si pamatuje data jen dokud je nepředá
- Žádné zábavné funkce a UI minimalistické
- Schválit/zamítnout propojení před přečtením
- Pro propojení musím znát přesný nick
- List propojených a zamítnutých
- U každého propojení vidím historii zpráv

## Screen

- Login/Registrace
- Nepřečtené
- Dopis
  - Tvorba
  - Psaní
- Kontakty
  - Schválené
    - Historie
  - Neschválené
    - Historie
- Nastavení

## Stack

- Language = TypeScript
- Framework = SolidJS
- CSS = solid-styled-components (goober)
- Validation = io-ts
- API = Cloudflare Workers
- Hosting = Cloudflare Pages
- Database = Macrometa (kv, document, graph)
